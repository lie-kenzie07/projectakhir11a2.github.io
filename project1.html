Project 1: Form & Tabel HTML

Deskripsi Project

Project ini merupakan implementasi awal dari pembelajaran HTML murni. Fokus utama dari tugas ini adalah menyusun struktur data menggunakan elemen <table> untuk menampilkan informasi secara tabular, serta merancang formulir interaktif menggunakan berbagai input seperti teks, dropdown, dan tombol submit melalui elemen <form>.
