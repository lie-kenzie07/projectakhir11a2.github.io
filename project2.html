Project 2: Eksplorasi UI/UX & CSS

Deskripsi Project

Project ini berfokus pada eksplorasi desain antarmuka (UI) dan pengalaman pengguna (UX). Melalui tugas ini, saya menerapkan penggunaan CSS Custom Properties (Variabel CSS) untuk menjaga konsistensi tema warna, serta memanfaatkan CSS Flexbox untuk mengatur tata letak elemen agar lebih dinamis, rapi, dan proporsional tanpa harus bergantung pada *float* atau *positioning* absolut.
